# CRM Quick Start — 5 Minutes to Your First Lead

## 🚀 Get Up and Running

### Step 1: Open CRM Pipeline
- Click **CRM Pipeline** in left sidebar
- See 7 empty columns: New, Contacted, Inspection Scheduled, Estimate Sent, Follow Up, Won, Lost

### Step 2: Create Your First Lead
- Click **New Lead** button (top right)
- Fill in:
  - **Name:** John Doe (required)
  - **Phone:** (555) 123-4567 (required)
  - **Service:** Water Damage
  - **Address:** 123 Main St
- Click **Create Lead**
- Lead appears in "New" column

### Step 3: Move Lead Through Pipeline
- **Hover** over lead card (shows grab cursor)
- **Drag** to next column (e.g., "Contacted")
- **Drop** to save instantly
- ✅ No save button needed, no refresh needed

### Step 4: Add Internal Notes
- **Click** lead card to open detail page
- Click **Notes** tab
- Type comment in text area
- Click **Comment** button
- Note appears instantly

### Step 5: Create a Task
- On lead detail page
- Click **Tasks** tab
- Type task title: "Schedule inspection"
- Click **Add Task**
- Task appears with checkbox to complete

### Step 6: Check Reminders
- Click **Reminders** in left sidebar
- See all active reminders grouped by type
- Click reminder to view lead
- Click **Dismiss** when done

## 💡 Key Features

### 🎯 Pipeline Stages (Left to Right)
```
New → Contacted → Inspection Scheduled → Estimate Sent → Follow Up → Won → Lost
```

### 💬 Notes
- Internal comments only (not customer-facing)
- Timestamped automatically
- Show on lead detail page

### ✅ Tasks
- Create action items
- Assign to team member
- Set due date
- Mark complete with checkbox

### 🔔 Smart Reminders
- Auto-generated when:
  - Estimate sent 3+ days ago
  - No customer response in 3+ days
  - Invoice unpaid
  - Job completed (request review)

### 📅 Activity Timeline
- Every action logged
- Stage changes
- Notes added
- Tasks created
- Files uploaded

## ⚡ Pro Tips

### Tip 1: Drag Constantly
- Update stage as conversations progress
- Each move is logged instantly
- Drag is the fastest way to stay current

### Tip 2: Notes Before Moving
- Add context note THEN drag to next stage
- Future you will thank current you
- Helps team stay on same page

### Tip 3: Create Tasks for Yourself
- "Call customer Tuesday"
- "Send estimate"
- "Follow up Friday"
- Keeps you accountable

### Tip 4: Monitor Reminders Daily
- Check **Reminders** page each morning
- Handle high-priority alerts (red)
- Dismiss when done
- Prevents dropped balls

### Tip 5: Team Collaboration
- Assign tasks to team members
- Add notes on lead activity
- Everyone sees full history
- No duplicate work

## 📊 What Each Stage Means

| Stage | Meaning | Next Action |
|-------|---------|-------------|
| **New** | Just created | Call/email customer |
| **Contacted** | Initial contact made | Schedule inspection |
| **Inspection Scheduled** | Date/time set | Run inspection |
| **Estimate Sent** | Quote given | Wait for response |
| **Follow Up** | No response yet | Call/email follow-up |
| **Won** | Customer approved | Create job/send review request |
| **Lost** | Customer declined | (Optional note why) |

## 🚨 Common Questions

**Q: Do I have to click Save?**  
A: No! Everything saves instantly. Just drag and move on.

**Q: Can customers see my internal notes?**  
A: No. Notes are internal only. Never shared with customer.

**Q: Who can see the leads?**  
A: Everyone in your company can see all leads. Tasks can be assigned to individuals.

**Q: How do I delete a lead?**  
A: Drag to "Lost" column or archive separately. (Full delete not available to preserve data.)

**Q: Can I customize the stages?**  
A: Currently fixed 7 stages. Contact support if you need changes.

**Q: What if I forget to follow up?**  
A: Reminders appear automatically. Check **Reminders** page daily.
